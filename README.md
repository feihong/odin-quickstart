# Feihong's Odin quickstart

## Install

```
sudo apt-get install llvm-11
git clone https://github.com/odin-lang/Odin
cd Odin
make
```

Add `alias odin=~/work/odin-quickstart/Odin/odin` to `~/.bashrc`.

## Commands

Build and run program

    odin run hello.odin
